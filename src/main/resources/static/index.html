FizzBuzz Game

The game is simple, a random number is generated and you have to go from the random number generated to the maximum number (100) and the numbers that are divisible by 3 put Fizz and those that are divisible by 5 put Buzz and if it is divisible by both put FizzBuzz.

Press the 'Generate' button to generate the random number and start the game.
